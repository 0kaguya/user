# Zed config

+ This folder contains parts of the user space zed config file, which is
  located at `.config/zed/settings.json`.

+ Schema is at `https://raw.githubusercontent.com/zed-industries/zed/refs/heads/main/assets/settings/default.json`.

+ AI-written lines should be write into `50-llm-gen.json`.

## Goals

+ Soft wrap at 80 characters.
+ Auto save when focus is off from editor.
